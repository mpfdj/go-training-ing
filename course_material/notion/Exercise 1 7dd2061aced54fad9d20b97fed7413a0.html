Exercise 1

Objective

Create a Go program that takes a string (a word or a sentence) as input and calculates the number of vowels in the string. As this is a simple program there is no need for creating an actual Go module, you can simply create a file with the .go suffix and run it using go run [filename].

Summary

  1. Use a for loop to iterate over each character in the string.
  1. For each character, check if it is a vowel (a, e, i, o, or u, case-insensitive).
  1. If it is a vowel, increase the count.
  1. Finally, print out the total count of vowels.

Example

Given the following input:

Go has built-in support for multiple return values.
This feature is used often in idiomatic Go.

The output should be 33 vowels.

Follow up

Instead of increasing the count by one for each vowel the value depends on the character encountered. The rules are as follows:

  • For a or A, increase counter by 2
  • For e or E, increase counter by 3
  • For i or I, increase counter by 5
  • For o/O or u/U, increase counter by 8
